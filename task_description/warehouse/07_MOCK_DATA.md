## Маппінг колонок Excel-імпорту → система

| Колонка Excel | Поле | Примітка |
|---|---|---|
| `Склад` | `warehouseId` | матч по назві або ID |
| `Категорія` | `categoryId` | матч по назві |
| `Підкатегорія` | `subcategoryId` | матч по назві в межах категорії |
| `Об'єкт витрат` | `expenseObjectId` | матч по `(warehouse, category, subcategory, name)` |
| `Тип витрат` | — | лише для валідації відповідності `expenseObject.costType` |
| `Рік` | `year` | |
| `Місяць` | `month` | число 1-12 або назва місяця (укр.) |
| `Сума` | `amount` | |
| `Коментар` | `comment` | опційно |

**Правила імпорту (узгоджено з бізнес-логікою §4.5 overview):**
- Рядок з `Тип витрат = Фіксована` і `Сума = 0` → помилка рядка
  (`ImportError`), не імпортується мовчки.
- Дублікат `(expenseObjectId, year, month)`, що вже існує → `updated`,
  не `imported` (upsert, як і в поточному `import_json.py`).
